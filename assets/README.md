# Project assets

## Icons

| File | Size | Use |
|------|------|-----|
| **`icon.png`** | 1024×1024 (square) | Favicon, app icon, GitHub repo icon |
| **`og.png`** | 1536×1024 (rect) | Open Graph / social preview (GitHub, link cards) |

### How to use

- **icon.png**: Reference in docs or web UI if you add a simple dashboard; can be used for `favicon.ico` or as the repo’s social preview in GitHub Settings → Social preview (upload there or point to a URL).
- **og.png**: Upload as “Social preview image” in GitHub repo **Settings → General → Social preview**, or use in any page’s `<meta property="og:image" content="...">` for link sharing.

If your square image is not 32×32 or 16×16, you can resize it for favicon; the rectangle one is typically used as-is for OG/social.
